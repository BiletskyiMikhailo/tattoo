Урок №:10 Початок Роботи

Матеріали/Обладнання для роботи

Матеріали:

  • Голки / Картриджі ( голки різного розміру та видів );
  • Фарба ( чорна або декілька кольорів );
  • Капси( одноразові ковпачки для наповнення фарбою );
  • Вазелін ( цей засіб не дозволяють фарбі «прилипати» до шкіри та спрощує її видалення. Під час сеансу майстер постійно протирає травмовану шкіру і вазелін знижує ризик подразнення та болючі відчуття );
  • Трансферний гель ( для переводу ескіза на шкіру );
  • Трансферний папір ( призначений для ручного переводу з малюнку в ескіз ,є також для машинного переводу призначений для спеціальних принтерів );
  • Ручка / Хірургічний фломастер ( який добре тримається на шкірі );
  • Одноразові простирадла ( для кушетки на якій розміщується клієнт );
  • Одноразові рукавиці;
  • Бандажні бинти ( намотується на держак тату машинки, або голку, для зручності в роботі );
  • Шпателі ( дерев’яні шпателі для набору вазеліну );
  • Медичні серветки ( необов’язково );
  • Плівка (« пластик» для обмотування робочої поверхні ,приладів та всього до чого доторкаєтесь,також для закриття тату в перерві між роботою );
  • Паперові рушники( для протирання всіх поверхонь, робочої зони на шкірі у клієнта );
  • Зелене мило ( для очищення шкіри під час сеансу, а також для обробки шкіри одразу ж після процедури);
  • Антисептик (для обробки шкіри клієнта, також для дезинфекції рук майстра );
  • Антисептик для поверхонь ( очищення робочої зони );
  • Крафт пакети ( для стерилізації держака );

    • Обладнання:

  • Світло ( лампа для комфорної роботи ,без світла не можливо побачити всіх нюансів і зробити якісне фото );
  • Холдер ( підставка під руку один із важливих складових роботи. Це допоміжне обладнання для найбільш комфортного розташування клієнта під час процедури );
  • Стіл (поверхня яка буде твоїм робочим місце );
  • Кушетка;
  • Стільчик для тебе і для клієнта;
  • Сухожар ( повітряна шафа для стерилізації );
  • Принтер;
